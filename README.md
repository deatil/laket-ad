## 广告系统


### 项目介绍

*  基于 `laket-admin` 后台管理框架的广告系统模块


### 安装使用

*  composer安装
    * `composer require lake/laket-ad`
*  本地安装
    *  手动将插件上传到 `/flashs/laket/laket-ad` 目录
*  最后在 `系统管理->闪存插件->本地` 安装添加的插件
